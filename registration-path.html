Registration Path

This page consists of general information about registration in bot.

Commands

  • /start - Start the bot and try to register.

  • /info - Get the worker main-information panel.

Before registration you can't use any of the commands besides /start.

Enter /start command to register in the bot.
After sending the command - your request to register will be sent to admin.

You'll get the response in a neck of time. Approximate waiting time: 2 hours

If you try to enter /start command once more - you'll get notification that you've already sent the request
